¿QUÉ ES CSS?

El CSS es el lenguaje que se utiliza para dar estilos a una página web, como colores, tamaños, márgenes y diseño en general.

¿QUÉ ES HTML?

HTML es el lenguaje que se utiliza para estructurar el contenido de una página web mediante etiquetas.

BOX MODEL

  • El box model organiza los elementos en forma de caja.
  • Padding: espacio interno entre contenido y borde.
  • Width: ancho del elemento.
  • Border: borde que rodea el elemento.
  • Margin: espacio externo entre elementos.

¿QUÉ ES DISPLAY?

  • Define cómo se muestran los elementos.
  • Block: ocupa todo el ancho.
  • None: oculta el elemento.
  • Inline: no salta de línea.

¿QUÉ ES FLEXBOX?

Flexbox es un sistema que permite organizar elementos de forma flexible dentro de un contenedor.

¿Cómo funciona?

  • Contenedor padre (flex container)
  • Elementos hijos (flex items)
  • Los elementos se acomodan automáticamente
