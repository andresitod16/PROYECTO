TEMAS VISTO EN CLASES

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que se utiliza para darle estilo a una página web. Permite cambiar colores, tamaños, fuentes, espacios y el diseño en general.

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje que se utiliza para crear la estructura de una página web mediante etiquetas como títulos, párrafos, imágenes y enlaces.

Box Model

el Box model es la forma en que CSS representa cada elemento como una caja rectangular. esta compuesto por varias partes que controlan el tamaño y el espacio.

  • Content: contenido del elemento.
  • Padding: espacio interno entre contenido y borde.
  • Border: borde que rodea el elemento.
  • Margin: espacio externo entre elementos.

Display

la propiedad display es el como se comporta un elemento en la pagina y el espacio que ocupa.

  • block: ocupa todo el ancho disponible.
  • inline: se mantiene en línea sin saltos.
  • inline-block: mezcla de ambos.
  • nonne: oculta el elemento.

Flexbox

flexbox es un sistema de diseño que permite organizar elementos dentro de un contenedor de forma flexible, alineándolos y distribuyéndolos fácilmente.

¿Como funciona?

  • Contenedor padre (flex container)
  • Elementos hijos (flex items)
  • Permite centrar y ordenar elementos fácilmente
