¿QUE ES CSS?

El css es el editor de estilos de la pagina es el encargado de darle color, medidas etc.

¿QUE ES HTML?

El HTML son multiples etiquetas que se utilizan para definir o estructurar el contenido de una pagina web.

BOXMODEL

  • Box model es el como se organiza los elementos de una pagina en forma de caja por medio del css. El box model tiene 4 caracteristicas que son:
  • Padding: Es el espacio interno que hay entre el contenido y el borde
  • Width: El width en otras palabras es una imagen a la cual se puede modificar mediante el css como width; 50px
  • Border: El border literalmente es lo que esta afuera del Padding y el width
  • Margin: Margin se utiliza para separarse de otros elementos
